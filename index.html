Cobbleverse Setup Guide

A step-by-step tutorial for installing and optimizing the Pokemon Adventure modpack via the Modrinth App.

1

Download the Launcher

Navigate to the Modrinth website and click the Download Modrinth App button to install the client.

2

Discover Content

Open the app and click the Compass Icon on the left sidebar to access the Modrinth library.

3

Install the Modpack

Search for "COBBLEVERSE" and select the green Install button. This handles the Fabric loader and all mod dependencies automatically.

4

Access Instance Settings

Once installed, go to your instance dashboard and click the Gear Icon in the top-right corner to open the configuration menu.

5

Allocate Memory

Navigate to Java and memory. Enable "Custom memory allocation" and set the slider to 8192 MB for optimal performance.

Note: 8GB is the recommended sweet spot for packs with 100+ mods and shaders.
